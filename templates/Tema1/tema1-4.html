    • ELEMENTOS DE UN PROGRAMA

    Constante

    En programación, una constante es un valor que no puede ser alterado durante la ejecución de un programa.

    Una constante corresponde a una longitud fija de un área reservada en la memoria principal, donde el programa almacena valores fijos.

    Por ejemplo:

    El valor de PI = 3.1416

    Variable

    Posición de memoria que posee valor, tipo, nombre (identificador), tamaño.

    Identificador de variable

  • Operaciones básicas: aritméticas, relacionales inicia con letra minúscula, sigue con letra, dígito, guión bajo ( _ ).
  • Si el identificador tiene más de una palabra la primera letra de cada palabra (a partir de la segunda) se escribirá con mayúscula
