¿QUÉ ES UN ALGORITMO?

Un algoritmo es una secuencia ordenada de pasos, exenta de ambigüedad, que permite la resolución de un problema determinado

Características de un algoritmo

  • Preciso: exento de ambigüedad.
  • Finito: si se sigue el algoritmo, se debe terminar en algún momento.
  • Definido: si se sigue dos veces el algoritmo con los mismo datos de entrada, la salida debe ser la misma.

    Herramientas para representar los algoritmos

    Pseudocódigo

    Una forma de describir un algoritmo en lenguaje humano que se encuentra entre el lenguaje natural y el código de programación real. Es útil para planificar y comunicar la lógica de un algoritmo antes de su implementación.

    Diagrama de flujo

    Gráficos visuales que utilizan símbolos y flechas para representar los pasos y la lógica de un algoritmo de manera fácil de entender y seguir.
